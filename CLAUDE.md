# Particle World - 粉末沙盒模拟游戏

## 项目概述
基于物理引擎的粉末沙盒模拟游戏，纯前端实现，运行在浏览器中。

## 技术栈
- TypeScript + Vite
- HTML5 Canvas（主渲染）
- 纯前端，无后端依赖

## 核心系统
- 粒子物理引擎（重力、流动、碰撞）
- 材质系统（沙、水、火、石、金属等）
- 化学反应系统（燃烧、融化、蒸发等）
- 用户交互系统（绘制、擦除、选择材质）

## 开发规范
- 每个功能一个原子 commit（feat/fix/refactor: 描述）
- 类型检查必须通过（tsc --noEmit）
- 构建必须成功（npm run build）
- 每次 commit 后 git push
- 代码要有清晰注释

## 迭代原则
每轮迭代完成后自我评估：
1. 当前版本完成了什么？
2. 有什么明显的问题或瓶颈？
3. 下一个迭代周期应该优先解决什么？
4. 是否需要调整技术方向或架构？

## 持续关注
- 🎮 可玩性：用户能否轻松创造有趣的场景？
- ⚡ 性能：模拟是否流畅？能否支持更多粒子？
- 🎨 表现力：材质视觉效果是否清晰美观？
- 🔬 真实性：物理行为是否符合直觉？
- 🧩 扩展性：添加新材质/新规则是否容易？

## 项目结构
```
src/
  main.ts          - 入口
  engine/          - 物理引擎核心
  materials/       - 材质定义
  ui/              - 用户界面
  utils/           - 工具函数
```
